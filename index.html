Project 2

You will be building an app that allows the user to play tic-tac-toe.

Requirements:
  • There should be one button for each square on the tic-tac-toe board, and pressing a button should play a move
  • Pressing a button on the tic-tac-toe board should make that button display either ‘X’ or ‘O’, depending on whose turn it is
  • There should be a label displaying whose turn it is
  • If someone has won (that is, there are either three O’s or three X’s in a row), the label should state who has won
  • If the game ends in a tie (that is, there are no empty squares and no one has won), the label should state that it is a tie
  • There should be a separate button that, when clicked, restarts the game
